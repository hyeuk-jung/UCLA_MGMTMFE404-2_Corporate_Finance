Executive Summary

The project researches on the effect of dividend payment announcement on the stock price. Here, we computed excess return over the S&P 500 Index constituents from 2010/01/01 to 2018/06/30, and concluded that the event of dividend payment has strong effect on the stock price and the impact size corresponds to the amount of the dividends.

Data Source

  1. S&P500 index constituents: S&P500 component tickers from 2010/01/01 to 2018/06/30
    (WRDS -> Compustat - Capital IQ -> Compustat -> North America - Daily -> Compustat Daily Updates - Index Constituents)
  2. Daily stock/market information:
    1) Holding period return;
    2) Holding period return without dividend;
    3) Dividend cash amount (which includes gives ex-dividend date information)
    4) S&P 500's Composite Index return
    (WRDS -> CRSP -> Annual Update -> Stock / Security Files -> CRSP Daily Stock)

Report

To test the stock performance in terms of excess rate return, when they are actually paying a dividend, we used the data from 2010/01/01 to 2018/06/30 and answered the following key question(s):

  1. Setting ex-dividend day as day 0, what happened to the stock return (minus the S&P500 on the same day) from days -20 through day +20, each day by day?
  2. How does each stock's rate of return relate to the amount of the dividend it actually paid?

For the calculation, we used total 15,306 complete set of 41 days results among 15553 dividend events.

The first plot below is calculated with daily holding period returns (including dividend) above S&P500 Index. The day before the ex-dividend day, the date the stock does not include the dividend right, the return decreased to -0.03%. This is not as much as the amount of dividend payout. Also on the ex-dividend day, even considering the included dividend payouts, the excess return increased more than expected. This implies:

  1. the distribution event gives a good signal to the market that the firm had good performance; and
  2. second, stock returns including dividends are theoretically correct.

The second plot uses the result calculated from daily holding period returns (excluding dividends) above S&P500 Index. The stock's returns remains similar throughout the 41-day period except for the ex-dividend day. On this day, the return plunges down to about -0.5%. This implies that once the dividend is paid out, the firm's value instantly reflects this event and the stock price is adjusted downward by the dividend amount and remains on that level unless other event happens.

##### Load libraries, S&P500 and Stock Data ---------------------
# Libraries
library(data.table)
library(magrittr)
options( warn = -1 )
# S&P500 constituents' 2010~2018.06 data (total 696 firms)
df1 <- read.csv("data_1014.csv", header = T) %>% as.data.table#; gc()
df2 <- read.csv("data_1518.csv", header = T) %>% as.data.table#; gc()
data <- rbind(df1, df2)
# Remove levels for HPR with and without dividends
data$RET <- as.numeric(as.character(data$RET)) #str(data)
data$RETX <- as.numeric(as.character(data$RETX))
data <- na.omit(data, cols = c("RET", "RETX")) # remove data with no HPR

##### Calculations ----------------------------------------------
# Calculate excess returns
data$ex.ret <- data$RET - data$sprtrn
data$ex.retx <- data$RETX - data$sprtrn
# Firm data
permno <- unique(data$PERMNO)
# Initialize 
ex.ret <- data.frame(matrix(NA, nrow=41, ncol=0))
ex.retx <- data.frame(matrix(NA, nrow=41, ncol=0))
# Append excess returns for each dividend distribution
for(i in 1:length(permno)) {# length: 696
  temp <- data[PERMNO == permno[i]] # extract one firm's data
  div <- which(temp$DIVAMT != 0) # get an index of the firm's distribution dates
  if(length(div) != 0) {
    for(j in 1:length(div)) {
      if(div[j]-20 <= 0) next
      ex.ret <- cbind(ex.ret, temp[(div[j]-20):(div[j]+20), "ex.ret"])
      ex.retx <- cbind(ex.retx, temp[(div[j]-20):(div[j]+20), "ex.retx"])
    }
  }
}
# Calculate average excess return for each day
ret_mean <- rowMeans(ex.ret, na.rm = TRUE)
retx_mean <- rowMeans(ex.retx, na.rm = TRUE)
# Plots and results
plot(x = -20:20, y = ret_mean, 'b', col = "blue", main = "Daily Average Excess Return w/ Dividend")

plot(x = -20:20, y = retx_mean, 'b', col = "red", main = "Daily Average Excess Return w/o Dividend")

round(ret_mean, 8)
